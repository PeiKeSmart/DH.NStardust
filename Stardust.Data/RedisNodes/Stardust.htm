Redis节点（RedisNode）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
ProjectId 项目 Int32 N 资源归属的团队
Name 名称 String 50
Category 分类 String 50
Server 地址 String 50 UQ 含端口
Password 密码 String 50
Version 版本 String 50
Mode 模式 String 50
Role 角色 String 50
MaxMemory 内存容量 Int32 N 单位MB
MemoryPolicy 内存策略 String 50 缓存淘汰策略
MemoryAllocator 分配器 String 50 内存分配器，低版本有内存泄漏
Enable 启用 Boolean N 停用的节点不再执行监控
ScanQueue 队列 Boolean N 自动扫描发现消息队列，默认true
WebHook 告警机器人 String 500 钉钉、企业微信等
AlarmMemoryRate 内存告警 Int32 N 内存告警的百分比阈值，百分之一
AlarmConnections 连接告警 Int32 N 连接数告警阈值
AlarmSpeed 速度告警 Int32 N 速度告警阈值
AlarmInputKbps 入流量告警 Int32 N 入流量告警阈值
AlarmOutputKbps 出流量告警 Int32 N 出流量告警阈值
CreateUser 创建人 String 50
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUser 更新人 String 50
UpdateUserID 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500


Redis数据（RedisData）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int64 PK N
RedisId Redis节点 Int32 N
Name 名称 String 50
Speed 速度 Int32 N 每秒操作数，instantaneous_ops_per_sec
InputKbps 入流量 Double N 单位kbps
OutputKbps 出流量 Double N 单位kbps
Uptime 启动时间 Int32 N 单位秒
ConnectedClients 连接数 Int32 N
UsedMemory 已用内存 Int32 N 单位MB
FragmentationRatio 碎片率 Double N 单位MB
Keys Keys数 Int64 N
ExpiredKeys 过期Keys Int64 N
EvictedKeys 驱逐Keys Int64 N 由于 maxmemory 限制，而被回收内存的 key 的总数
KeySpaceHits 命中数 Int64 N 只读请求命中缓存
KeySpaceMisses Miss数 Int64 N 只读请求未命中缓存
Commands 命令数 Int64 N
Reads 读取数 Int64 N
Writes 写入数 Int64 N
AvgTtl 平均过期 Int64 N 平均过期时间，单位毫秒
TopCommand 最忙命令 String 200
CreateTime 创建时间 DateTime
TraceId 追踪 String 50 最新一次查看采样，可用于关联多个片段，建立依赖关系，随线程上下文、Http、Rpc传递
Remark 备注 String 500
