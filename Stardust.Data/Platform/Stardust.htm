星系项目（GalaxyProject）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
Name 名称 String 50 UQ
Enable 启用 Boolean N
TenantId 租户 Int32 N 该项目所属租户，实现隔离
ManagerId 管理者 Int32 N
Nodes 节点数 Int32 N
Apps 应用数 Int32 N
Sort 排序 Int32 N
IsGlobal 全局 Boolean N 该项目的节点可以允许其它项目下应用选用
WhiteIPs IP白名单 String 50 符合条件的来源IP才允许访问，支持*通配符，多个逗号隔开
BlackIPs IP黑名单 String 50 符合条件的来源IP禁止访问，支持*通配符，多个逗号隔开
NewServer 新服务器 String 50 该项目下的节点自动迁移到新的服务器地址
CreateUserId 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500
